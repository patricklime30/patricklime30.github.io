AI-Powered Psychotest System

A web-based psychotest platform designed to assess users using the Big Five Personality Traits (Openness, Conscientiousness, Extraversion, Agreeableness, Neuroticism). The system supports multiple question types (Likert scale, multiple choice, short answer) and uses AI (Ollama) to evaluate answers and generate a PDF personality report.



Key Features

  • Various Question Types like: Multiple choice, Likert scale, and short answer.

  • Integrated with Ollama to evaluate answers and calculate trait scores.

  • Proctoring tools with Face detection, Noise detection and Screen capture every 10 seconds.

  • Admins can review face detection logs, screen captures, and noise activity to determine if the participant may have cheated.

  • Generates a downloadable personality report after the test.

  • Manage questions, participants, and view reports.

Project information

  • Category: Web App
  • Backend: Laravel (PHP framework) and Ollama (AI Model)
  • Frontend: Blade templates, HTML, CSS, JavaScript, Tailwind CSS, face-api.js, MediaRecorder API
  • Database: MySQL
